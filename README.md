# Hapi REST API starter

Based on ["How to create a REST API with Hapi"](http://blog.webkid.io/how-to-create-a-rest-api-with-hapi/) .

## Installation

```
$ npm install
```

## Start Server

```
$ npm start
```

## What do you get?

A simple word REST API:

```
http://localhost:8080
  GET    /word                         
  POST   /word                         
  GET    /word/{id}                    
  POST   /word/{id}                    
  PATCH  /word/{id}                    
  DELETE /word/{id}  
```
